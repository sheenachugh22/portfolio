I combine structured PM practice with hands-on delivery credibility so teams commit to realistic dates, risks surface early, and releases land with clear quality and handover.

Side projects · Real tasks I wanted to speed up

Publishing prep, Etsy listings, and a calmer investing routine

Independent builds · 2025–2026

These are not demos for their own sake—they each started from a repetitive weekly workflow: too many tabs, too much copying, too easy to skip the review step. The pattern is the same in all three: you steer, the app remembers, AI drafts, you approve, then you get files or a screen you can act on.

Amazon / Etsy book and printable prep

Use case: Go from “I want a low-content book or printable this week” to a shortlist of ideas, then one chosen concept with title ideas, keywords, interior outline, sample page text, and prompts you can hand to Canva or KDP—without storing any marketplace passwords in the tool.

Tech, briefly: web app in TypeScript/React (Next.js), data saved locally in SQLite, AI through Cursor’s API by default with OpenAI as an option.

Etsy digital product builder (early layout)

Use case: Same rhythm as above but aimed at digital downloads on Etsy—a screen flow for the creator, a small API behind it, and prompt text kept in files so wording is easy to tune.

Tech, briefly: Next.js UI, Node/Express API, prompts in a shared folder.

Personal investment advisor

Use case: “I have a bit to invest and I already watch certain ETFs or stocks—what deserves attention this week?” The app keeps holdings, a watchlist, a dated recommendation history, and simple charts so a weekly review takes minutes instead of an afternoon.

Tech, briefly: Python service (FastAPI) for data and rules, React dashboard, SQLite, Docker Compose for a consistent setup.
